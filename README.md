# Siphelele Mfusi — Business Website

Personal business website for **Siphelele Mfusi**, offering CV & Resume Design, Interview Preparation, and Website Development services based in Utrecht, KwaZulu-Natal.

---

## 📁 Project Structure

```
website/
├── index.html    → All page content & structure
├── style.css     → All styling, layout & animations
├── script.js     → Navbar, scroll effects, counters & form
└── README.md     → This file
```

---

## 🚀 How to Run Locally

1. Download or clone this repository
2. Open the `website/` folder
3. Double-click `index.html` — it opens in your browser
4. No installation or server needed

---

## 🌐 How to Deploy on GitHub Pages (Free Hosting)

1. Go to [github.com](https://github.com) and sign in
2. Click **New Repository** — name it e.g. `my-website`
3. Upload all 3 files: `index.html`, `style.css`, `script.js`
4. Go to **Settings → Pages**
5. Under **Source**, select `main` branch → click **Save**
6. Your site will be live at:
   ```
   https://lindasphelele.github.io/my-website
   ```

## ✏️ How to Edit Content

| What to change | Where to find it |
|---|---|
| Your name, bio, location | `index.html` → About section |
| Prices | `index.html` → Pricing section |
| Phone & email | `index.html` → Contact section & Footer |
| Colours & fonts | `style.css` → `:root` variables at the top |
| Stats (CVs, Websites) | `index.html` → `data-target="50"` values in Hero |
| Add/remove services | `index.html` → Services & Pricing sections |
